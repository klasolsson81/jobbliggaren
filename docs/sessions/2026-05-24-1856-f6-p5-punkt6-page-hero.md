# F6 P5 Punkt 6 — page-hero + brand-empty på inre sidor (HANDOVER-v4)

## Sammanfattning

Ren FE-uppdatering per `HANDOVER-v4`-designspec. `.jp-pagehero`-suit (page-hero
efter `.jp-hero` landing-mönster) + `.jp-empty--brand` (brand-empty efter
`.jp-empty` neutralt-mönster) införd och applicerad på `/oversikt`,
`/ansokningar` och `/cv`. Inga BE-ändringar, inga migrations, inga nya
dependencies. Pushad på origin/main, tag `v0.2.68-dev` triggar Vercel-deploy.

## Mål

1. Lägg `.jp-pagehero` (page-hero på inre sidor) + `.jp-empty--brand` (brand-empty)
   som ny CSS-suit ovanpå befintliga `.jp-hero`/`.jp-empty`.
2. Applicera på 3 sidor (`/oversikt`, `/ansokningar`, `/cv`) per HANDOVER-v4 §2.1.
3. Säkerställ edge-to-edge-rendering (undvik `.jp-shell-transitional-container`-
   wrappning för nya hero-sidor).

## Vad levererades

5 filer ändrade + 2 review-rapporter, **+427 / -110** rader, 1 commit (`f28236a`).

### CSS-suit (`globals.css`)

`.jp-pagehero` + tillbehör (`__title`, `__sub`, `__meta`, `__cta`, `__aside`)
efter `.jp-hero`-blocket. `.jp-empty--brand` + tillbehör (`__title`, `__sub`,
`__cta`) efter `.jp-empty`-blocket. Verbatim från HANDOVER-v4 §2.1 plus
`:focus-visible { outline: 2px solid #fff }` per HANDOVER §4.2-rekommendation
(navy-ytorna behöver vit ring för 3:1-kontrast — civic-utility A11y).

### `/oversikt` (`components/oversikt/oversikt-page.tsx`)

Title-block (h1 + sub) → `.jp-pagehero`. TodayCard placerad i
`.jp-pagehero__aside` (rätt-justering på desktop, wrap under på mobil).
`.jp-container jp-page` behållen för Notiser + Sammanfattning under hero.

### `/ansokningar` (`app/(app)/ansokningar/page.tsx`)

`.jp-pagehero` + brand-empty: Pipeline-kicker, "Inga ansökningar ännu" som
empty-rubrik, två CTAs (`Lägg till ansökan` + `Sök jobb`). `Search`-ikon
importerad från `lucide-react`. Empty visas när ingen ansökan finns.

### `/cv` (`app/(app)/cv/page.tsx`)

`.jp-pagehero` + brand-empty: CV-varianter-kicker, "Inga CV ännu" som
empty-rubrik. Layout omstrukturerad från Tailwind `flex flex-col gap-6` till
`.jp-container jp-page` för pattern-paritet med `/ansokningar` och `/oversikt`.

### App-shell (`components/shell/app-shell.tsx`)

`/cv` tillagd i `V3_NATIVE_ROUTES`. Utan tillägget skulle `/cv` wrappats i
`.jp-shell-transitional-container` → bryter `.jp-pagehero` navy edge-to-edge.
HANDOVER-v4 §3 nämnde det villkorligt ("om wrappar något extra") men ej
explicit /cv-tillägg.
